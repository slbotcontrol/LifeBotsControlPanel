# TELEPORT_OFFER_ACCEPT

Commands bot to accept a teleport offer

## Variables

The following table shows input values (you send them with the API call) and returned output values.

| Variable | Required | Description               |
|:-------- |:-------- |:------------------------- |
| str      | yes      | ---                       |
| id       | yes      | Session ID                |

## Example

```lsl
llMessageLinked(LINK_SET, TELEPORT_OFFER_ACCEPT, "", "d11fa786-c342-4785-8e1b-ac70a17a9aac");
```
