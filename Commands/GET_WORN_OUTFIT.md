# GET_WORN_OUTFIT

Returns a list of all items currently worn by the bot.

## Variables

The following table shows input values (you send them with the API call) and returned output values.

| Variable | Required | Description               |
|:-------- |:-------- |:------------------------- |
| str      | yes      | ---                       |
| id       | yes      | ---                       |

## Example

```lsl
llMessageLinked(LINK_SET, GET_WORN_OUTFIT, "", "");
```
