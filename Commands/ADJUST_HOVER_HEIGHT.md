# ADJUST_HOVER_HEIGHT

Sets the hover height of the bot avatar.

## Variables

The following table shows input values (you send them with the API call) and returned output values.

| Variable | Required | Description               |
|:-------- |:-------- |:------------------------- |
| str      | yes      | Hover height              |
| id       | yes      | ---                       |

## Example

```lsl
llMessageLinked(LINK_SET, ADJUST_HOVER_HEIGHT, "0.2", "");
```
