# REGION_INFO

Returns detailed information about the current simulator/region.

## Variables

The following table shows input values (you send them with the API call) and returned output values.

| Variable | Required | Description               |
|:-------- |:-------- |:------------------------- |
| str      | yes      | ---                       |
| id       | yes      | ---                       |

## Example

```lsl
llMessageLinked(LINK_SET, REGION_INFO, "", "");
```
