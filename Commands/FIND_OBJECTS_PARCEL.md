# FIND_OBJECTS_PARCEL

Searches for objects in the current parcel.

## Variables

The following table shows input values (you send them with the API call) and returned output values.

| Variable | Required | Description               |
|:-------- |:-------- |:------------------------- |
| str      | yes      | ---                       |
| id       | yes      | ---                       |

## Example

```lsl
llMessageLinked(LINK_SET, FIND_OBJECTS_PARCEL, "", "");
```
